# 吴子杰的个人网站

> 在互联网公海修筑的一座私人小岛 🏝️

**[zijiewu.eu.cc](https://zijiewu.eu.cc)**

## 关于

纯手工 HTML + CSS + JS 单页，没有框架，没有构建工具，就像一个数字花园。

## 彩蛋

- 🎮 **Konami Code** — `↑↑↓↓←→←→BA`
- 🖱️ **点击计数器** — 右下角那个小圆点，试试看
- 🗺️ **四角探索** — 页面四个角落各点几下
- 💡 **项目卡片** — 每张卡片都能点
- 🔮 **更多...** — 等你发现

## 技术栈

- 纯 HTML / CSS / JS
- Three.js (via CDN)
- GitHub Pages
- 零构建、零框架
